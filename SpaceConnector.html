Space Connector Overview

Gameplay

"Space Connector" is a prototype designed to merge a traditional connect game with a space shooter. It features fast-paced space combat integrated with a unique connecting mechanic. This game was created as part of my university game design course.

Main takeaways

Prototyping

The primary focus of this project was to prototype various mechanics and determine which ones worked best. Starting with a theme, I developed the idea of connecting enemies and built upon it. While the game is still quite basic at this stage, I believe the core gameplay loop has proven to be fun. With additional content, this game has the potential to be truly great.

Documentation

As part of our game design course, we were required to thoroughly document our design process, detailing all the decisions we made along the way in Miro. Additionally, we had to create a very short and comprehensive Game Design Document, which you can find here: SpaceConnectorGDD.
