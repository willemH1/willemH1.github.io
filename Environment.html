Stylized Unity Environment Overview

Process

The building was created using a modular asset approach, with assets modeled and unwrapped in Blender, textured in Substance Painter, and then imported into Unity. The foliage was created using Unity Shader Graph, particles like leaves and butterflies were generated with Unity's Particle System, and the trees were made using Tree.It. The scene is game-ready and optimized for performance, with LODs, static batching, and GPU instancing significantly reducing draw calls.
